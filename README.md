CruzBE
======
### How to Started CruzBE

####In terminal
#####Test it on browser
```
$ ionic serve
```
#####Test it on IOS simulator
```
// Currently facebook login has an issue due to the update of Firebase
$ ionic bulid ios
$ ionic emulate ios
```
#####Test it on Android simulator
```
//After connecting your Android phone to your computer
$ ionic run android
```
